# OpenPTV working folder

Courtesy of University of Maryland

## Setup without PyPTV installation
1. Download the zip file and unzip
2. Install `uv`
3. In the folder open your terminal and run: `uvx --python 3.11 pyptv .`

## Installation of PyPTV 
Explained on our website https://www.openptv.net
